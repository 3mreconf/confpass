# ConfPass - Password Manager

Modern ve güvenli şifre yöneticisi uygulaması. Windows için Tauri tabanlı masaüstü uygulaması ve Chrome/Edge tarayıcı eklentisi.

## Özellikler

- AES-256 şifreleme ile güvenli şifre saklama
- Otomatik şifre doldurma
- Güçlü şifre oluşturucu
- Yerel şifreleme - verileriniz bulutta saklanmaz
- Tarayıcı eklentisi desteği
- Modern ve kullanıcı dostu arayüz

## Gereksinimler

- Windows 10/11
- Chrome veya Microsoft Edge tarayıcısı (extension için)

## Kurulum

1. [Releases](https://github.com/3mreconf/confpass/releases) sayfasından en son sürümü indirin
2. Installer'ı çalıştırın
3. Uygulamayı açın ve ana şifrenizi oluşturun
4. Tarayıcı eklentisini [Chrome Web Store](https://chrome.google.com/webstore)'dan yükleyin
